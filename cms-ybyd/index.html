Dashboard

Setelah login, masuk ke menu dashboard. Lalu buat batch baru, dengan mengklik menu batch



Batch

Setelah masuk ke menu batch, bikin batch baru dengan mengklik create batch



Setelah masuk ke dalam create batch, isi form sesuai yang dibutuhkan lalu klik save



Order / List PO

Setelah batch baru dibuka, sekarang tahap membuat orderan atau PO dengan cara klik create order



Setelah masuk ke halaman create order, sekarang tahap membuat orderan atau PO berdasarkan contoh kasus diatas



Setelah orderan dibuat, sekarang tahap mengirim invoice dengan cara klik create invoices



Sekarang Pilih po yang ingin dikirim invoice, masukkan ongkir jika diperlukan



Sekarang screenshoot invoice, lalu kirim invoice ke yang customer



Jika Customer sudah membayar invoice, sekarang bikin transaksi dengan klik menu transaksi



Setelah masuk ke menu transaction, klik create transaction



Lalu isi semua form yang disediakan, jangan lupa untuk mengisi order.



Jika po sudah bayar semua di tiap batch, sekarang saatnya export report tiap batch ke excel yang sudah disiapkan. Masuk ke menu batch



Tekan icon mata untuk melihat detail batch



Pilih Export



Lalu klik copy table



Buka spreadsheet yang udah ada, pilih sheet yang DATA ENTRY. Lalu paste ke excel, rapihkan lagi formatnya
